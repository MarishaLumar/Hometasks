Список команд по роботі з терміналом

  • $ git branch- Список іменованих гілок коммітів із зазначенням обраної гілки.
  • $ git add . | --all | -A- каже: "Принесіть зміни в віддалений репозиторій туди, де зберігаю свій власний код".
  • $ git commit- Редагування комміту.
  • $ git init- Створення репозиторію.
  • $ git add [file-name] - Додавання змін.
  • $ git rm [файл] - Переміщення та видалення версій файлів репозиторію.
  • $ git clean | -dxf - Очищення репозиторію.
  • $ git add [file-name] - Додавання змін.
  • $ git push - Відправка змін.
