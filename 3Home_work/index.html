Список команд по роботі з терміналом

  1. $ git branch- Список іменованих гілок коммітів із зазначенням обраної гілки.
  2. $ git add . | --all | -A- каже: "Принесіть зміни в віддалений репозиторій туди, де зберігаю свій власний код".
  3. $ git commit- Редагування комміту.
  4. $ git init- Створення репозиторію.
  5. $ git add [file-name] - Додавання змін.
  6. $ git rm [файл] - Переміщення та видалення версій файлів репозиторію.
  7. $ git clean | -dxf - Очищення репозиторію.
  8. $ git add [file-name] - Додавання змін.
  9. $ git push - Відправка змін.
